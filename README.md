# 🎮 ActivityBuilder · 游戏活动页模板工具

一个**纯前端、零依赖构建**的游戏活动页可视化搭建工具。把常用的活动模块（Banner、签到、抽奖、任务、礼包、排行榜、累充进度、规则…）直接拖到画布上即可成稿，并支持一键导出长图、保存/导入 JSON 模板。

## ✨ 功能特性

- **拖拽搭建**：左侧模块库 → 中间画布，所见即所得
- **15 个游戏向预设模块**：
  - Banner 头图 / 章节标题 / 倒计时 / 每日签到 / 九宫格抽奖
  - 任务列表 / 礼包兑换 / 排行榜 / 累充进度 / 充值档位
  - 活动规则 / 按钮组 / 文本块 / 图片 / 间距
- **6 套游戏主题配色**：赛博紫、节庆红、深海蓝、魔幻绿、日落橙、暗黑金
- **属性面板编辑**：文字、颜色、尺寸、列表项增删、图片 URL/本地上传
- **画布操作**：选中、上移/下移、复制、删除、画布内拖拽排序
- **手机外壳预览**：3 种画布宽度（375 / 414 / 750）+ iPhone 外壳样式
- **一键导出**：长图（PNG, 2x 高清）+ JSON 模板（可重新导入二次编辑）
- **预览模式**：去掉编辑态，查看真实视觉

## 🚀 使用方式

无需任何安装步骤：

```bash
# 方式一：直接双击打开
open index.html

# 方式二：起一个本地静态服务（推荐，避免本地图片跨域）
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

> ⚠️ **图片导出注意**：如果你在 Banner / 图片模块里填写了**外链图片 URL**，导出长图时可能因跨域 CORS 被拦截。建议改为**上传本地图片**（属性面板里的"选择文件"按钮），或将图片放到本工具同源目录下。

## 🧩 添加自定义模块

在 `modules.js` 的 `MODULES` 对象里新增一项即可：

```js
mySection: {
  icon: '🎯',
  name: '我的模块',
  data: () => ({ title: '默认标题' }),
  render: (d) => `<div class="my-section">${d.title}</div>`,
  propsSchema: [
    { key: 'title', label: '标题', type: 'text' }
  ]
}
```

支持的 `propsSchema.type`：`text`、`textarea`、`number`、`color`、`bool`、`image`、`list`（带 `itemFields`）。
